Snapshot Week Configuration

Sets the current working week for data entry. Dashboard viewing is separate — this controls which week accepts new data.

When the week advances, current snapshot data is frozen and a new snapshot week begins. All unsaved changes in Data Entry will be carried over. Previous snapshots remain viewable in read-only mode.
